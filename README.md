# 超像素分割Streamlit 網頁端
## 依賴庫
- streamlit
- PIL
- stqdm
- opencv-contrib-python
- numpy
## 安裝
**要把requirements.txt 中的 opencv-contrib-python-headless 改成opencv-contrib-python**
> pip install -r requirements.txt
## 使用方法
> streamlit run app.py
